action API path Method URL Remarks
1 Get all notes ~/api/getNotes/ GET Get all notes
2 Get a specific Note ~/api/getOneNote/<int:serial> GET Get the 1st Note use note's serial no.
3 Add a Note ~/api/addNotes/ POST Add a Note send JSON obj with 'title' and 'desc'
